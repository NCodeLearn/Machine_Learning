# 📂 Kaggle Dataset Checklist for ML & Image Processing

Daily Commitment: 2 Hours  
Duration: 3–4 Months  

---

## Phase 0: Python & Data Foundations (Weeks 1–2)
**Goal:** Practice NumPy & Pandas with simple datasets.  
- [Titanic: Machine Learning from Disaster](https://www.kaggle.com/c/titanic) → Beginner-friendly, great for Pandas practice.  
- [Iris Dataset](https://www.kaggle.com/uciml/iris) → Classic dataset for classification.  
- [MNIST Handwritten Digits](https://www.kaggle.com/c/digit-recognizer) → Image data in CSV format, perfect for NumPy array manipulation.  

---

## Phase 1: Classical ML + Image Basics (Weeks 3–5)
**Goal:** Apply Scikit-Learn + OpenCV.  
- [Fashion MNIST](https://www.kaggle.com/datasets/zalando-research/fashionmnist) → Clothing images for classification.  
- [Dogs vs. Cats](https://www.kaggle.com/c/dogs-vs-cats) → Binary image classification.  
- [Leaf Classification](https://www.kaggle.com/c/leaf-classification) → Shape-based image recognition.  

---

## Phase 2: Deep Learning Foundations (Weeks 6–9)
**Goal:** Build CNNs with Keras.  
- [CIFAR-10](https://www.kaggle.com/c/cifar-10) → 10-class image dataset (airplanes, cars, animals).  
- [Sign Language MNIST](https://www.kaggle.com/datamunge/sign-language-mnist) → Hand gesture recognition.  
- [ASL Alphabet](https://www.kaggle.com/grasshoppermouse/american-sign-language-dataset) → Larger sign language dataset for CNN practice.  

---

## Phase 3: Advanced Computer Vision (Weeks 10–14)
**Goal:** Transfer learning, detection, segmentation.  
- [COCO Dataset (subset)](https://www.kaggle.com/datasets/awsaf49/coco-2017-dataset) → Object detection tasks.  
- [Pascal VOC 2012](https://www.kaggle.com/datasets/huanghanchina/pascal-voc-2012) → Segmentation dataset.  
- [Chest X-Ray Images (Pneumonia)](https://www.kaggle.com/paultimothymooney/chest-xray-pneumonia) → Medical image classification.  

---

## Phase 4: Deployment & Testing (Weeks 15–18)
**Goal:** End-to-end project + deployment.  
- [Dogs vs. Cats Redux](https://www.kaggle.com/c/dogs-vs-cats-redux-kernels-edition) → Use transfer learning + deploy API.  
- [Plant Seedlings Classification](https://www.kaggle.com/c/plant-seedlings-classification) → Real-world agricultural dataset.  
- [Face Mask Detection](https://www.kaggle.com/datasets/andrewmvd/face-mask-detection) → Object detection + deployment project.  

---

## 🚀 Suggested Project Progression
1. **MNIST → Fashion MNIST → CIFAR-10** (build CNNs).  
2. **Dogs vs. Cats → Transfer Learning (ResNet/VGG)**.  
3. **Custom dataset (Face Mask Detection or Plant Seedlings)** → Deploy with Flask/FastAPI.
